🌐 NetData API

NetData — это инструмент для работы с геоданными, организациями и пользователями, включающий систему верификации по почте и полнотекстовый поиск (FTS5).

# 🛠 Быстрый старт
1. Подготовка окружения

Для изоляции зависимостей проекта рекомендуется использовать виртуальное окружение:
### Установка утилиты virtualenv (если не установлена)
```sh
python -m pip install virtualenv
```

### Создание окружения
```sh
python -m venv .venv
```


### Активация (Windows PowerShell)
```sh
.\.venv\Scripts\Activate.ps1
```
### Активация (Windows CMD)
```sh
.\.venv\Scripts\activate.bat
```


2. Установка зависимостей
```sh
pip install fastapi uvicorn loguru requests rich pydantic
```
    Примечание: Библиотеки sqlite3, csv, pathlib, json, re, math, smtplib и secrets являются стандартными для Python и не требуют отдельной установки.


🚀 Запуск и тестирование
Основной сервер

Запустите API сервер командой:
```sh
python ./main.py
```

Тестирование функционала

Скрипт client.py проверяет полный цикл работы: регистрацию, создание и проверку токена.
```sh
python ./client.py
```
    [!IMPORTANT]
    Перед запуском тестов:

        Рекомендуется удалить файл netdata.db (он пересоздастся автоматически).

        Отключите VPN, если используете почтовый сервис (SMTP), чтобы избежать блокировок.

        Проверьте настройки лимитов и токенов в config.json.

        Логи работы доступны в директории /logs.

📍 Геокодирование

Для проверки запросов предусмотрено две программы. Важно: перед запуском вставьте API-токен, полученный из письма, в код скриптов.
Прямое геокодирование (Поиск по адресу)

Используйте searchtest.py.
```sh
python ./searchtest.py
```

Обратное геокодирование (Поиск по координатам)

Используйте searchcoords.py.
```sh
python ./searchcoords.py
```
📂 Структура проекта

    main.py — Точка входа в приложение.

    db.py — Логика работы с SQLite и индексами FTS5.

    api.py — Роуты FastAPI и бизнес-логика.

    config.json — Конфигурация SMTP, токенов и уровней логирования.

    data_with_coords.csv / organizations.csv — Исходные данные для базы.
